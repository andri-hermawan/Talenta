## Yang Harus di Install
1. NodeJs
2. Loopback Framework
3. Angular 2++
4. Mysql

## Cara Menjalankan
Quick start options:
- Clone the repo.
- Import Database.
- Npm Install
- Open Terminal:
  cd Talenta/talenta-server
  npm install && node.
- Open Tab Terminal:
  cd Talenta/talenta-client
  npm install && npm start
